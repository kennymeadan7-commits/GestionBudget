# GestionBudget

Site web simple de gestion de budget personnel.

## Fonctionnalités

- Ajout de revenus et de dépenses
- Calcul automatique des totaux (revenus, dépenses, solde)
- Filtre par mois
- Objectif de budget mensuel avec barre de progression
- Répartition des dépenses par catégorie
- Suppression de transactions
- Sauvegarde locale via `localStorage`

## Lancer le projet

Aucune dépendance n'est nécessaire.

1. Ouvrir `index.html` dans un navigateur
2. Commencer à enregistrer vos transactions
